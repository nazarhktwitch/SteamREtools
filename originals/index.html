Custom Hubcap Key Set your own key if the built-in one stops working
How to get a key (quick guide)
  1. Open hubcapmanifest.com
  2. Click Continue with Discord
  3. Go to API section -> click Generate Key
  4. Copy the key (starts with smm_) and paste it above

Keys reset every week - generate a new one when downloads stop working.
